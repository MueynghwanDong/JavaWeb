element selector

  • 전체 선택자

    1. 페이지에 있는 모든 요소를 대상으로 스타일을 적용하려 할 때 사용
    2. * { margin:0 ; padding:0 ; }

  • 태그 선택자

    1. 문서 안의 특정 태그에 스타일이 모두 적용되도록 할 때 사용
    2. p { font-size : 12px ; font-family : “돋움”; }

  • 클래스 선택자

    1. 태그에 class 이름을 지정하여 특정 class에만 적용되게 할 때 사용
    2. .redaccent { color : red ; }
    3. 어디든 .redaccent 라는 class가 선언된 태그에 모두 적용

  • id 선택자

    1. class와 유사하게 특정 id로 지정된 태그들에 적용할 때 사용
    2. . 대신 # 기호 사용
    3. id는 문서상에서 하나만 나와야 함

  • 하위 선택자(모든 하위 자손에게 전달됨)

    1. 적용 대상을 한정시키려 할 때
    2. 상위요소 하위요소 { 속성: 값; …}
    3. section p{ color: blue;}
    4. section 안에 있는 p에 대해서만 전경색을 파란색으로
    5. #container ul { border:1px dotted black;}
    6. container id가 선언된 태그 안에 있는 모든 ul 태그들에 border 적용

  • 자식 선택자(1촌에게만 전달됨)

    1. 하위 선택자와 유사하나 직계 자식에게만 전파됨
    2. 상위요소 > 하위요소 { 속성: 값; …}}
    3. section > p{ color: blue;}
    4. section 바로 밑에 있는 p에 대해서만 전경색을 파란색으로

  • 인접형제 선택자

    1. 같은 부모 아래 동일한 depth로 등록된 태그 중 처음 만나는 태그 선택
    2. h1+p { text-decoration:underline; }
    3. h1과 같은 depth에 있는 태그 들 중 처음 만나는 p에 대해 underline 처리}

  • 형제 선택자

    1. 같은 부모 아래 동일한 depth로 등록된 태그 선택
    2. h1~ p { text-decoration:underline; }
    3. h1과 같은 depth에 있는 태그 들 중 모든 p에 대해 underline 처리

  • 그룹 선택자

    1. 여러 요소에 한꺼번에 스타일 적용
    2. 요소1, 요소2, …{ 속성:값;…}
    3. a, p {color:purple; }
    4. a와 p 태그에 대해 purple 색 적용
